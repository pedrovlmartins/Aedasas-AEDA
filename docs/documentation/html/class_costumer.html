Descrição detalhada

Classe derivada de User: Costumer. Classe que contem a informacao generica de um cliente que pode usar os servicos oferecidos pela agencia de viagens.

Trip Costumer::getTrip ( string  destination) const

Funcao que, fazendo uso do metodo de pesquisa sequencial, verifica se o cliente ja visitou o destino passado como argumento. Caso a resposta seja afirmativa, a funcao devolve o indice no vector dessa viagem. Caso contrario, devolve -1.
